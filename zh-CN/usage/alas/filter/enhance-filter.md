# 强化舰船过滤器

## 过滤项

| 舰种 | 描述 | |
| ---- | ---- | ---- |
| dd | 驱逐 | Destroyer |
| ss | 潜艇 | Submarine |
| cl | 轻巡 | Light Cruiser |
| ca | 重巡 | Heavy Cruiser |
| bb | 战舰 | Battleship |
| cv | 航母 | Aircraft Carrier |
| repair | 维修舰 | Repair Ship |
| others | 其他 | - |

## 例子

```text
dd > cl > ca > bb > ss > cv
```
